Projects

The organization's repository can be found at https://github.com/AKI-Club/.

Tools

  • akiarc – AKI archive handler
  • akitext – AKI text archive handler
  • emuscripts – Scripts for emulators with scripting support
  • menubg – Convert PNG ↔ Menu Background (WM2000, VPW2, No Mercy)
  • vpw2romid – Virtual Pro-Wrestling 2 ROM identifier
  • vpwspal – VPW Studio palette conversion tools
